About these samples the honest bit

These are portfolio pieces, built to show what properly done CRM work looks like when it’s taken seriously end to end: data you can trust, a system shaped to the business, and a client who always knows what’s happening. They’re not slideshows or mockups; they’re real, dependency-free web apps that run entirely in your browser.

Every person, company, message and number is fictional. Each sample keeps all its data in a small, readable file and calculates every screen from it on the spot, which is why the dashboards, reports and record pages always agree with each other. The best way to judge the work is to click into any of the three and try to break it.

Prefer reading code to clicking buttons? The whole thing is open: github.com/rounaksingh890/crm-portfolio — plain HTML, CSS and JavaScript, no build step, one readable data file per demo.

Not affiliated with or endorsed by HubSpot or any CRM vendor. The samples just follow interface conventions people already know. Built as an independent demonstration.
